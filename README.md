## 👋 About This Bot

This is the official Telegram Bot for **[IVANNIKOV.PRO](https://ivannikov.pro)** — a team of senior Blockchain, Web3, and GameFi developers.

The bot serves as an automated portfolio showcase and lead generation tool, featuring:
- 📱 **Interactive Portfolio & Services menu**
- 🧙‍♂️ **Multi-step Lead Collection Wizard** (sends structured leads directly to admins)
- 📄 **On-the-fly CV PDF Generation** (English/Russian)
- 🔗 **Built-in Referral System**
- ⚡ **Inline Promo Cards** (type `@ivannikovProBot` in any chat to share our services)

## 🏗 Architecture & Stack

The source code for this bot is maintained privately in our enterprise monorepo infrastructure to ensure maximum security and stability.

**Tech Stack used to build this bot:**
- **Runtime:** Node.js (ESM)
- **Language:** Strict TypeScript
- **Framework:** Telegraf (with `Scenes.Stage` for wizards)
- **PDF Generation:** PDFKit + JS-YAML
- **Monorepo System:** Turborepo + pnpm workspaces
